功能說明

“音樂遊戲培訓” app是一款練習聲音遊戲/節奏遊戲的應用。
你可以通過播放視頻查看音符，也可以直接觸摸視頻模擬演奏。

添加播放視頻

有兩種方法可以添加視頻在應用程序中播放。

設備內或谷歌驅動器

在設備上或在谷歌驅動器中播放視頻。
下面是如何添加它。

  1. 打開 “視頻列表” 畫面
  2. 點擊屏幕右下方的 “+” 按鈕
  3. 從附加建議列表中選擇 “設備內 + Google Drive”
  4. 選擇你要添加的視頻
  5. 輸入您的註冊信息並點擊 “加法” 按鈕

如果您在這一步驟後返回 “視頻列表 “屏幕，您將看到您添加的視頻列表。

播放YouTube視頻

YouTube是谷歌的視頻分享服務。存在別人上傳的音樂遊戲視頻。
下面是如何添加它。

  1. 打開 “視頻列表” 畫面
  2. 點擊屏幕右下方的 “+” 按鈕
  3. 從附加建議列表中選擇 “YouTube”
  4. 從YouTube網站上打開你要添加的視頻頁面
  5. 點擊視頻頁面右下角的 “添加視頻” 按鈕
  6. 輸入您的註冊信息並點擊 “加法” 按鈕

如果您在這一步驟後返回 “視頻列表 “屏幕，您將看到您添加的視頻列表。

因為它不下載YouTube視頻，所以每次在這個應用上播放視頻都會產生流量。

偽遊戲玩法

當您在播放視頻時，點擊視頻不會暫停。
所以，你可以玩一個偽音樂遊戲。

還有很多其他功能可以幫助你練習。

  • 調整視頻播放速度
    您可以在0.25-2.00倍速度範圍內調整播放速度
  • 調整視頻播放的開始/結束位置
    你可以設置遊戲範圍的開始和結束，以專注於遊戲中最難的部分
  • 放大/縮小並調整視頻的位置
    可以進行縮放和定位，使視頻適合設備的分辨率
  • 支持視頻屏幕的縱向和橫向方向
    您可以在縱向或橫向模式下固定您的設備方向
  • 在播放过程中，可倒退/快进几秒钟的快捷键
    不需要暂停和操纵时间

以低雙倍速度播放

有一個降低視頻播放速度的功能。
"音樂太快了，我聽不懂！" 試著用 0.25 倍速播放，然後逐漸加快 0.05 倍速。
嘗試在低雙倍速度下理解樂譜，並在逐漸提高速度的同時進行偽演奏。

收藏播放位置

有一個功能可以為視頻播放的開始/結束位置做書籤。
你可以為每個數字設置一個開始/結束的位置，這樣你就可以快速改變你要練習的部分的播放位置。

放大/縮小和移動視頻的位置

掐進/掐出可以放大/縮小，滑動可以移動位置。
你可以將游戲的截圖導入應用程序。在觀看圖像時，調整圖像的透明度。

調整音頻延遲

如果視頻和音頻不同步，音頻延遲可以 0.01 秒為增量進行調整。
如果視頻本身有音頻延遲，或者您使用的藍牙耳機有音頻延遲，請進行調整。
